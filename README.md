# Scholar-Arena
Scholar Arena is a gamified study platform that transforms learning into an engaging competition. Students can earn XP, maintain study streaks, climb leaderboards, and challenge themselves through study arenas designed to make academic growth fun and motivating.

Built with Flask, HTML, and CSS, Scholar Arena aims to help students stay consistent, track their progress, and turn studying into an exciting journey rather than a boring task.

🚀 Features (Planned)
- XP and Level System
- Study Streak Tracking
- Global Leaderboards
- Study Challenges and Arenas
- User Profiles
- Achievement Badges
- Premium Features
